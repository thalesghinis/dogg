
Sincronizar os arquivos do Servidor para o Computador:
1. Abrir o Shell do XAMPP ou o Git Bash
2. Na tela de comandos (SHELL/Bash) acessar a pasta:
executar o comando "CD C:/xampp/htdocs/{nomeProjeto}"
3. Executar o comando "git status" para verificar o status da pasta.
4. Executar o comando "git pull" para sincronizar os arquivos existentes entre o servidor e o seu computador.
	Resposta: "Already up to date."
5. Comece a trabalhar nos arquivos.
